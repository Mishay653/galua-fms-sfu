# galua-fms-sfu

Данная методичка посвящена изучению теории Галуа и её применению к вычис-
лению групп Галуа конкретных алгебраических уравнений. Материал охватывает ос-
новные определения, теоремы, алгоритмы вычислений и содержит обширную коллек-
цию задач различной сложности. Предназначена для школьников старших классов и
студентов младших курсов, изучающих современную алгебру.

Методический материал по теории Галуа   Автор: Светличный М.Ю.   ФМШ СФУ, 2025
