# Вычисление групп Галуа алгебраических уравнений

Методический материал по теории Галуа  
Автор: Светличный М.Ю.  
ФМШ СФУ, 2025

## Аннотация

Данная методичка посвящена изучению теории Галуа и её применению к вычислению групп Галуа конкретных алгебраических уравнений. Материал охватывает основные определения, теоремы, алгоритмы вычислений и содержит обширную коллекцию задач различной сложности.

Предназначена для школьников старших классов и студентов младших курсов, изучающих современную алгебру.

В репозитории находятся PDF-файл методички, решения задач и дополнительные примеры.
