# galua-fms-sfu

## Аннотация

Данная методичка посвящена изучению теории Галуа и её применению к вычислению групп Галуа конкретных алгебраических уравнений. Материал охватывает основные определения, теоремы, алгоритмы вычислений и содержит обширную коллекцию задач различной сложности. Предназначена для школьников старших классов и
студентов младших курсов, изучающих современную алгебру.

Методический материал по теории Галуа
